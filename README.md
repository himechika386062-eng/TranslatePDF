■ 概要
Deepl APIを使ってPDFファイルのテキストを翻訳するアプリです。

=================================================
■ 起動方法
=================================================

① ZIPファイルを右クリック
② 「すべて展開」を選択
③ フォルダを開く
④ 「PDFTranslator.exe」をダブルクリック

=================================================
■ 初回設定
=================================================

★初回起動時に「DeepL APIキー」を入力してください。
※APIキー入力後にウィンドウが閉じるので、再度「PDFTranslator.exe」をダブルクリックするとアプリが使えるようになります。

DeepL APIキーのユーザー登録は以下のようなページを参考にしてください。
	https://auto-worker.com/blog/?p=5030
登録は無料でできますが、個人と紐づけられるため、クレジットカードの情報などが必要です。

なお、DeepL APIの仕様上、フリープランのユーザーは翻訳できる文字数が月50万文字に制限されています。
APIキーひとつについての上限ではなく、アカウントあたりの上限数なので注意してください。
また、文字数の使用状況はDeepLのユーザーページで確認できます。

APIキーが取得出来たらコピー＆ペーストしてキーをアプリに入力して下さい。
二回目以降は不要ですが、アプリ内で使用するキーの変更は可能です。


=================================================
■ 操作方法
=================================================

★翻訳したいPDFファイルをアプリ上に追加します。「+PDF」ボタンあるいはドラッグで追加できます。
★追加したPDFはカーソルを合わせると「×」ボタンで消去できます。
★翻訳先の言語を選択します。
★対訳表示するかをチェックします（任意）。
★「翻訳開始」ボタンで翻訳を開始します。
★「キャンセル」ボタンで翻訳を中止します（それまでに翻訳された分の文字数は消費されます）。
※「APIキー更新」ボタンでAPIキーを更新できます。

=================================================
■ 注意事項
=================================================

・インターネット接続が必要です。
・Windowsの警告が出る場合があります。
  → 「詳細情報」→「実行」を押してください。
・セキュリティソフトにブロックされる場合があります。
・このアプリの配布は個人間の直接的な伝達に限って自由に行ってもらって構いません。
